# Tablero Farmacia

App en Streamlit para generar graficos de movimientos por codigo de medicamento.

## Ejecutar

```bash
source .tab_fcia/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

## Tablero ROP

```bash
source .tab_fcia/bin/activate
pip install -r requirements.txt
streamlit run app_rop.py
```

## Datos

- `data/movimientos.csv` (requerido)
- `data/rop_sugerido.csv` (opcional)

Tambien podes subir los CSV desde la barra lateral.
